**hong-kong-open-map** is an open-source library that makes it incredibly easy to embed beautiful, high-precision interactive maps of Hong Kong on any website — completely free and without Google Maps pricing.

Built on official Hong Kong Government vector tiles from the Lands Department (geodata.gov.hk), it delivers accurate, up-to-date geography with crisp rendering at all zoom levels.

Designed for everyone:
- Non-developers can embed a fully customizable map with a single <iframe>
- Vanilla JS users get a lightweight script with no dependencies
- React/Next.js developers get a fully-typed <HongKongOpenMap /> component

Key features:
- Official Hong Kong Government data (more accurate than OSM in many areas)
- Light and elegant dark themes
- Easy marker/pin support with custom colors and HTML popups
- Responsive and mobile-friendly
- No API keys, no usage limits, no tracking
- Proper attribution to the HKSAR Government Lands Department

A proud open-source alternative that puts Hong Kong’s geospatial data in the hands of the community.
