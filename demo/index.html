Hong Kong Open Map

Free, open-source map component for Hong Kong applications.
Uses official Lands Department vector tiles. No API key required.

香港開放地圖 · 免費開源的香港地圖組件

Installation

# npm
npm install hong-kong-open-map
# yarn
yarn add hong-kong-open-map
# pnpm
pnpm add hong-kong-open-map
# bun
bun add hong-kong-open-map

CDN / Direct HTML Usage

index.html 
<!-- 1. Include MapLibre GL -->
<link rel="stylesheet" href="https://unpkg.com/maplibre-gl@4/dist/maplibre-gl.css">
<script src="https://unpkg.com/maplibre-gl@4/dist/maplibre-gl.js"></script>

<!-- 2. Include hong-kong-open-map -->
<script src="https://unpkg.com/hong-kong-open-map/dist/vanilla.umd.js"></script>

<!-- 3. Create map container -->
<div id="map" style="width: 100%; height: 500px;"></div>

<!-- 4. Initialize map -->
<script>
  const map = new HKMap('map', {
    center: [114.1694, 22.3193],
    zoom: 12,
    language: 'zh',
    theme: 'light'
  });
</script>

React / TypeScript

App.tsx 
import { HKMap, HKMarker } from 'hong-kong-open-map';
import 'hong-kong-open-map/style.css';

export default function App() {
  return (
    <HKMap
      center={[114.1694, 22.3193]}
      zoom={12}
      language="zh"
      theme="light"
      showZoomControl
      showScaleControl
    >
      <HKMarker
        position={[114.1694, 22.3193]}
        title="Victoria Harbour 維多利亞港"
      />
    </HKMap>
  );
}

API Reference

HKMapProps

Property Type Default Description
center [number, number] [114.17, 22.32] Map center [lng, lat] in WGS84
zoom number 11 Initial zoom level
minZoom number 10 Minimum zoom level
maxZoom number 15 Maximum zoom level
theme 'light' | 'dark' | 'auto' 'light' Map color scheme
language 'zh' | 'en' 'zh' Label language
showLabels boolean true Show place and street labels
labelScale number 1.0 Label size multiplier (0.5–2.0)
showZoomControl boolean true Show +/− zoom buttons
showScaleControl boolean false Show scale bar
showNavigationControl boolean false Show compass
showFullscreenControl boolean false Show fullscreen button
width string | number '100%' Container width
height string | number '400px' Container height
onLoad (map) => void — Callback when map loads
onClick (event) => void — Callback on map click

HKMarkerProps

Property Type Default Description
id string required Unique identifier
position [number, number] required Marker coordinates [lng, lat]
title string — Tooltip text
popup string | ReactNode — Popup content
color string '#000000' Marker color
